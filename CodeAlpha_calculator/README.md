# Neumorphic Calculator

A web-based calculator built with a modern, soft UI (Neumorphism) aesthetic. This project was developed as an internship task to demonstrate DOM manipulation and CSS styling capabilities.

## ✨ Features

-   **Standard Operations:** Addition, subtraction, multiplication, and division.
    
-   **Neumorphic Design:** Soft, extruded plastic look using advanced CSS shadows.
    
-   **Keyboard Support:** Fully functional using keyboard number pad and operators.
    
-   **History Tracking:** Displays the previous operand and operation.
    
-   **Edit Controls:** Includes 'All Clear' (AC) and 'Delete' (DEL) functionality.
    

## 🛠️ Technologies Used

-   **HTML5** - Structure
    
-   **CSS3** - Styling & Animations
    
-   **JavaScript (ES6)** - Logic & DOM Manipulation
