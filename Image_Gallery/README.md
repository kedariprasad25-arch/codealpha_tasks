#  Image Gallery

A responsive, modern web-based image gallery featuring category filtering and a lightbox view.

## ✨ Features

-   **Category Filtering:** Sort images dynamically by categories (Nature, City, Animals, Tech).
    
-   **Lightbox View:** Click any image to open it in a full-screen, focused modal with a dimmed background.
    
-   **Gallery Navigation:** Seamlessly switch between photos inside the lightbox using "Next" and "Previous" buttons.
    
-   **Modern UI:** Designed with a "Glassmorphism" aesthetic, featuring translucent elements, neon accents (`#00d2ff`), and smooth hover animations.
    
-   **Responsive Grid:** Automatically adapts the layout to fit different screen sizes.
    

## 🛠️ Technologies Used

-   **HTML5** - Semantic structure.
    
-   **CSS3** - Grid layout, Flexbox, and CSS Variables for theming.
    
-   **JavaScript (ES6)** - DOM manipulation for filtering and modal logic.
    

## 🚀 How to Run

1.  Clone or download this repository.
    
2.  Open `index.html` in any modern web browser.
    
3.  Click the filter buttons to sort images or click an image to view the lightbox.
