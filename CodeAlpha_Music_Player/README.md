# 🎵 Vinyl Music Player

A retro-styled, web-based music player featuring a rotating vinyl animation and a modern glassmorphism UI.

## ✨ Features

-   **Interactive Controls:** Play, pause, skip, and rewind functionality.
    
-   **Vinyl Animation:** The record disk rotates while playing and stops when paused.
    
-   **Playlist System:** A side panel displays the song queue; clicking a track instantly plays it.
    
-   **Progress & Volume:** Clickable progress bar for seeking and a dedicated volume slider.
    
-   **Modern UI:** Built with a "Glassmorphism" aesthetic, featuring a blurred backdrop, gradient background (`#4203a9` to `#904e95`), and neon accents (`#00ffcc`).
    

## 🛠️ Technologies Used

-   **HTML5** - Structure and Audio API.
    
-   **CSS3** - Flexbox, Keyframe Animations, and Backdrop Filter.
    
-   **JavaScript (ES6)** - DOM manipulation, Audio events, and Playlist logic.
    

## 🚀 How to Run

1.  Clone or download this repository.
    
2.  Open `index.html` in your browser to start listening.
